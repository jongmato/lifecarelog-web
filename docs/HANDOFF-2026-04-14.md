## 진행 중인 작업 (백그라운드 에이전트 4개)

| 에이전트 | 작업 | 대상 파일 |
|----------|------|-----------|
| harden | 접근성 P0/P1 수정 | contact-dialog, hero, contact-section-v2, nav, footer, badge, story |
| optimize | 성능 P1/P2 수정 | hero(FloatingOrbs CSS전환), nav(scroll throttle), framer-motion→motion/react |
| normalize-distill | 테마 통일 + 안티패턴 제거 | globals.css(card 웜화이트, gradient/glow 제거), footer(하드코딩 색상) |
| adapt | 반응형 P1/P2 수정 | services grid 중간단계, 모바일 nav 추가, dvh, text-wrap balance |

**⚠️ 주의**: 4개 에이전트가 동일 파일을 동시 수정할 수 있어 충돌 가능. 완료 후 빌드 확인 필수.

### 추가 개선 (우선순위순)
1. **GmarketSans 로컬화**: 외부 CDN → /fonts/ 로컬 woff2
2. **logo-text.png 교체**: 흰 배경 버전 필요 또는 SVG 벡터 텍스트 로고 제작
3. **Impeccable 재감사**: `/audit` 재실행하여 점수 확인 (목표: 16+/20)
4. **OG Image 생성**: SEO용 Open Graph 이미지
5. **Cloudflare Pages 배포**: `pnpm build` → CF Pages

### 미해결 이슈
- `logo-text.png` 검은 배경 → `mixBlendMode: 'screen'`으로 임시 처리 중. 투명 배경 PNG 또는 SVG 필요
- `motion/react` 패키지 교체 — package.json에서 `framer-motion` → `motion` 변경 필요 여부 확인
- 모바일 햄버거 메뉴 — adapt 에이전트가 구현 중
